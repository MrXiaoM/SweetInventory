# SweetInventory

Minecraft 菜单插件

## 简介

基于 [PluginBase](https://plugins.mcio.dev/elopers/base/gui/) 菜单模块制作的菜单插件。使用类似于以下示例的直观排版格式
```yaml
inventory:
  - '框框框框框框框框框'
  - '上页页页页页页页下'
  - '框框框框框框框框框'
```
图标可以添加显示条件、点击条件、点击操作等等，支持 `左键`、`右键`、`Shift+左键`、`Shift+右键`、`Q键`、`Ctrl+Q键` 等点击方式。

详见 [MCIO Plugins](https://plugins.mcio.dev/docs/inventory/intro) 上的文档。
